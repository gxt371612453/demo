1.一般法务提供的相关合同文件为doc或docx文件（其中不包含占位符信息），我们需要将拿到的文件进行调整，即将下表中占位符信息填充到合同文本相关位置（.docx文件）。

2.将处理过的文件通过手动执行demo实例进行初步转换 -> html文件 (此时的html文件，内容格式或许不符合要求，需要手工进行微调)

具体操作：

* 处理合同的docx文件的辅助工具，使用前提：手动修改docx文件

* 仅仅用于处理docx文件，如果提供了doc文件，请手工修改为docx类型

* 其中@@@ 标记该段落要居左（顶行）

* 其中### 标记段落为标题段落，需要居中

* 其中$$$ 标记段落为居右段落

* 正常缩进不需要特殊标记

* (视具体合同而定，可以不处理，待生成html后再手动调整)特殊处理签章内容的 %%% ，将标记放置在签章方中间，用于替换空格标签 eg: 甲方 %%% 乙方 -> 甲方（签章）   乙方（签章）

  

   文件转换demo操作入口:

  demo
